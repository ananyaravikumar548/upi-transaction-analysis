## Overview
This project analyzes Unified Payments Interface (UPI) transaction data in India to uncover
trends in transaction volume and value over time.

## Objectives
- Understand growth patterns in UPI usage
- Compare transaction volume vs transaction value
- Perform clean and structured Exploratory Data Analysis (EDA)

## Dataset
- Source: NPCI (Public UPI transaction data)
- Format: Excel (monthly aggregated data)

## Tools & Technologies
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
